justify-content is a property used in Flexbox to align flex items along the main axis (horizontal axis for a row, vertical axis for a column).

gap is a shorthand property in both Flexbox and Grid layouts that defines the gap between flex items or grid rows/columns.

align-items is used in Flexbox to align flex items along the cross axis (perpendicular to the main axis).

flex-direction sets the direction of the main axis in a flex container.

flex-wrap controls whether the flex container should wrap its items onto multiple lines if there's not enough room in the container.
